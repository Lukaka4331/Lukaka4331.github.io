Learning to Segment Every Thing

- 𝑀𝑎𝑠𝑘X𝑅𝐶𝑁𝑁

Abstract

  1. 標註新內容的成本很高
  2. 提出一個新的部分監督方法來傳遞權重(weight)的函數
  3. 大量的類別標註bounding box
    • dataset : Visual Genome dataset
  4. 只有小部分標註Segmentation
    • dataset : COCO dataset

Introduction

  1. 預測一個前景的segmentation mask 對於每個偵測物件
  2. 典型的instance segmentation 只有包括大約100個物件種類
  3. 這種限制的一個原因是最新技術實例分割算法需要有強力的監督，而這種監督可能是有限且昂貴的收集新類別
  4. 標註預測框更豐富也更便宜
  5. paper介紹一個新的部分監督instance segmentation任務,提出一個新穎的遷移式學習方法來解決
  6. 部分監督instance segmentation任務如下:
    • 給定一組感興趣的類別，一小部分具有instance mask 標註，而其他類別只有邊界框標註
    • training data 是帶有強烈的標註(有mask)和只有位置的標註(only bounding box - predict box )
  7. 主要效益 部分監督vs弱監督 training 建立large-scale的instance segmentation model 利用現有資料集的兩種型態:
    • 使用邊框標註在很多類別,比如:Visual Genome - dataset
    • 使用instance mask標註在小類別上,比如 : COCO - dataset
  8. 解決部分監督的instance segmentation 提出一個新穎的遷移式學習方法建立在Mask R-CNN
